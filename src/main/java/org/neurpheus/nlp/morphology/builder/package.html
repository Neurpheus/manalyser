Morhological Analyser build classes.

Package Specification

This package contains classes which build a morphological analyser. {@link org.neurpheus.nlp.morphology.builder.MorphologicalAnalyserBuilder} performs a building process. The {@link org.neurpheus.nlp.morphology.builder.MorphologicalAnalyserByuildHelper} is a help class for th ebuilder. The {@link org.neurpheus.nlp.morphology.builder.InflectionTreeBuilder} is also helping class which builds an inflection tree. The {@link org.neurpheus.nlp.morphology.builder.InflectionNodePrinter} prints inflection tree nodes during the inflection tree logging.
